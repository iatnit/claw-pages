🦞 一、什么是"小龙虾"？5分钟

一句话解释

OpenClaw（小龙虾）是一个开源的 AI 私人助手框架，它不只是聊天，还能真正帮你做事——发邮件、整理文件、管日程、搜资料、写报告，甚至操控你的电脑。

跟 ChatGPT 的区别

❌ ChatGPT

  • 只能在网页/App里聊天
  • 聊完关掉就忘了
  • 不能操作你的电脑
  • 不能连接你的工具
  • 你问它才答

✅ 小龙虾

  • 在微信/飞书/钉钉里跟你对话
  • 有记忆，越用越懂你
  • 能读写文件、操控浏览器
  • 能连接邮箱、日历、笔记等
  • 能主动帮你做事（定时任务）

三层架构（简单理解）

大模型是大脑，小龙虾是身体，Skills是手脚。没有Skills的小龙虾 = 聪明但没有手脚的哲学家。

🎓 二、老师为什么需要它？10分钟

国际学校老师的真实工作痛点

📝 场景1：英文作业批改 & 评语生成
IGCSE/A-Level 学生的英文作文、Essay、Lab Report 发给小龙虾 → 自动标注语法错误、结构问题 → 生成个性化评语（英文） → 你只需审核确认。一篇 Essay 评语从 15 分钟缩到 2 分钟。
🌍 场景2：中英文教案 & 课件翻译
"把这份中文教案翻译成英文" / "帮我把这个英文课件的要点整理成中文给家长" → 秒出结果，比 Google 翻译准确得多，而且懂教育术语。
📊 场景3：学期报告 & 评估表自动生成
"根据这学期的教学记录，帮我生成 Report Card 的评语模板" / "帮我写 IGCSE predicted grade 的依据说明" → 批量生成个性化学生评估，格式直接可用。期末不再加班到凌晨。
📧 场景4：邮件处理 & 家校沟通
自动帮你检查邮箱 → 家长邮件立即通知 → 帮你草拟中英文回复 → "这封邮件帮我用礼貌但坚定的语气回复，告诉家长孩子需要加强阅读"。
📖 场景5：备课资料搜集
"帮我找 IGCSE English Language 关于 persuasive writing 的教学活动设计" / "A-Level Chemistry 有机化学的实验教案参考" → 同时搜索中英文资源 → 整理成可用的教案框架。
📅 场景6：日程管理 & 会议提醒
每天早上自动推送：今天几节课、什么时候开 staff meeting、有没有 parent conference → 不再错过任何安排。
🔧 场景7：日常效率工具箱
查天气、翻译文件、整理会议纪要、生成活动海报文案、帮外教翻译中文通知… 工作+生活全覆盖。
💡 跟 ChatGPT 最大的区别：ChatGPT 需要你打开网页主动去用。小龙虾直接在你的微信里，像同事一样跟你协作。而且它有记忆——上周让它帮你查的资料，这周它还记得。你可以在微信群里直接 @它。

🎭 四、配置：给它一个灵魂10分钟

核心原则：所有配置都可以通过跟小龙虾聊天来完成！
不需要自己找文件、改代码。直接说"帮我创建一个 SOUL.md 文件"就行。

必配的两个文件

1. SOUL.md — 它的性格（三观）

决定 AI 助手的说话方式和行为准则。针对国际学校老师，推荐双语效率型模板：

# SOUL.md

你是小助，一个高效、直接、靠谱的 AI 助手。

## 你的性格
- 回答问题直奔主题，不说废话
- 能一句话说清楚的就不用三段话
- 做事先做再汇报，不要反复问"要不要我做"
- 可以有自己的想法和建议

## 你的语言能力
- 默认用中文回复
- 如果我发英文给你，用英文回复
- 翻译任务要求准确、地道，注意教育类术语
- 帮我写英文邮件/评语时，语气要专业但温暖

## 你的规矩
- 对外操作（发邮件、发消息）必须先问我
- 对内操作（读文件、查资料、写草稿）大胆做
- 出了错误主动告诉我

## 你不要做的事
- 不要每次都说"好的！我来帮你处理"，直接做
- 不要把简单的事情讲得很复杂

2. USER.md — 你的个人档案

# USER.md
- 名字：王老师（改成你自己的名字）
- 身份：国际学校教师
- 学科：IGCSE/A-Level English（改成你的学科）
- 时区：Asia/Shanghai
- 语言：中英双语，日常中文，教学材料经常需要英文
- 沟通偏好：简洁直接
- 常用场景：备课、批改Essay、写Report Card评语、
           家校沟通邮件、翻译通知、整理会议纪要

配置前

配置后

🛡️ 八、安全须知5分钟重要

三件必做的事

  1. 不要把端口开放到公网
    确认地址是 127.0.0.1:18789（只有自己能访问），不要改成 0.0.0.0
  2. 保护好 API Key
    API Key = 密码。不要截图发群里、不要上传到任何地方
  3. 定期更新
    openclaw update — 至少每两周一次
⚠️ 全球已有 42,000+ 个小龙虾实例因配置不当暴露在公网，近 1,800 个泄露了 API 密钥和聊天记录。不要成为其中之一。

执行审批机制

小龙虾做高风险操作时会先问你：

  • Allow once（允许一次）— 不确定时选这个
  • Always allow（永远允许）— 确认安全后选这个
  • Deny（拒绝）— 感觉不对就拒绝

🙋 九、Q&A + 动手环节20分钟

常见问题预备

Q: 这跟 ChatGPT 到底有什么不同？

A: ChatGPT 是"你问它答"的网页聊天。小龙虾是真正的执行助手——它能操作你的文件、连接你的工具、出现在你的微信/Telegram里、主动帮你做事、还有记忆。你上周让它帮你改的作业模板，这周它还记得。

Q: 要花多少钱？

A: 软件本身免费开源。AI模型用国内免费额度（Kimi/通义千问/MiniMax）够日常使用。如果用云平台一键部署，基本零成本。想用更强的模型（Claude/GPT-4），大约每月几十块。

Q: 我完全不懂技术，能用吗？

A: 能！用云平台方案（MiniMax/Kimi），1分钟就能用上。而且大部分配置都可以直接跟小龙虾聊天完成，不需要你写代码。

Q: 数据安全吗？会不会泄露学生信息？

A: 本地部署的数据全在你自己电脑上。但要注意：发给小龙虾的内容会通过AI模型API处理。建议：不要发送学生身份证号、护照号等隐私数据。用它来批改作业、备课、翻译、管日程是安全的。

Q: 能帮我翻译和写英文邮件吗？

A: 这是它最擅长的！中英互译、写英文家长信、批改英文作文、翻译学校通知… 比 Google Translate 强很多，它懂教育场景的语境。

Q: Mac 和 Windows 都能用吗？

A: 都可以。本地安装 Mac/Windows/Linux 全支持。云平台方案更简单，跟系统无关。
