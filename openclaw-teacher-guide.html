🦞 一、什么是"小龙虾"？5分钟

一句话解释

OpenClaw（小龙虾）是一个开源的 AI 私人助手框架，它不只是聊天，还能真正帮你做事——发邮件、整理文件、管日程、搜资料、写报告，甚至操控你的电脑。

跟 ChatGPT 的区别

❌ ChatGPT

  • 只能在网页/App里聊天
  • 聊完关掉就忘了
  • 不能操作你的电脑
  • 不能连接你的工具
  • 你问它才答

✅ 小龙虾

  • 在微信/飞书/钉钉里跟你对话
  • 有记忆，越用越懂你
  • 能读写文件、操控浏览器
  • 能连接邮箱、日历、笔记等
  • 能主动帮你做事（定时任务）

三层架构（简单理解）

🧠 大脑
AI大模型
（理解+推理）
→
🦞 小龙虾
Gateway网关
（调度中心）
→
🔧 技能Skills
具体工具
（搜索/邮件/文件…）

大模型是大脑，小龙虾是身体，Skills是手脚。没有Skills的小龙虾 = 聪明但没有手脚的哲学家。

🎓 二、老师为什么需要它？10分钟

教师专属使用场景

📝 场景1：自动批改作业反馈
把学生作文/作业内容发给小龙虾 → 它帮你写详细评语 → 指出优点和改进方向 → 你只需审核确认
📊 场景2：自动生成周报/学期总结
"帮我根据本学期的教学日志，生成一份教学工作总结" → 格式化输出，直接可用
📖 场景3：备课助手
"帮我查找关于《背影》这篇课文的教学设计参考" → 搜索+总结+整理 → 给你可借鉴的教案框架
📅 场景4：日程管理 + 会议提醒
每天早上自动推送：今天有几节课、几点开会、有什么待办 → 连接日历后全自动
📧 场景5：邮件/通知管理
自动帮你检查邮箱 → 重要邮件立即通知 → 不重要的整理成摘要 → 还能帮你草拟回复
🔬 场景6：课题研究资料搜集
"帮我搜集最近一年关于'项目式学习在初中数学中的应用'的论文和案例" → 联网搜索+总结要点
🏠 场景7：生活效率
查天气、查火车票、提醒家庭事务、翻译文件、辅导孩子功课… 工作之外照样好用
💡 关键卖点：ChatGPT 需要你打开网页主动去用，小龙虾直接在你的微信/飞书/钉钉里，像同事一样跟你协作。而且它有记忆——上周让它帮你查的资料，这周它还记得。

🅲 本地安装要点（给想折腾的老师）

系统要求：
  • Windows 10+ / macOS / Linux 均可
  • Node.js ≥ v22（安装脚本会自动装）
  • 5GB以上磁盘空间
  • 稳定网络

一键安装命令（Windows PowerShell）：

# Windows 先解除脚本限制（重要！否则报红字错误）
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

# 然后安装
npx openclaw@latest

Mac/Linux：

curl -fsSL https://openclaw.ai/install.sh | bash
⚠️ Windows常见坑：90%新手会遇到"系统禁止运行脚本"的红字报错。一定要先运行 Set-ExecutionPolicy RemoteSigned -Scope CurrentUser 再安装。

🎭 四、配置：给它一个灵魂10分钟

核心原则：所有配置都可以通过跟小龙虾聊天来完成！
不需要自己找文件、改代码。直接说"帮我创建一个 SOUL.md 文件"就行。

必配的两个文件

1. SOUL.md — 它的性格（三观）

决定 AI 助手的说话方式和行为准则。对老师来说，推荐效率型模板：

# SOUL.md

你是小助，一个高效、直接、靠谱的 AI 助手。

## 你的性格
- 回答问题直奔主题，不说废话
- 能一句话说清楚的就不用三段话
- 做事先做再汇报，不要反复问"要不要我做"
- 可以有自己的想法和建议

## 你的规矩
- 对外操作（发邮件、发消息）必须先问我
- 对内操作（读文件、查资料、写草稿）大胆做
- 出了错误主动告诉我

## 你不要做的事
- 不要每次都说"好的！我来帮你处理"，直接做
- 不要把简单的事情讲得很复杂

2. USER.md — 你的个人档案

# USER.md
- 名字：王老师
- 身份：初中语文教师
- 时区：Asia/Shanghai
- 沟通偏好：简洁直接，中文回复
- 常用场景：备课、批改作业、写总结、管日程

配置前 vs 配置后

配置前

你："帮我查明天天气"

AI："好的！我很高兴能帮助您查询天气信息！明天的天气预计是……"

配置后

你："帮我查明天天气"

AI："王老师，明天上海：多云转晴，18-25°C，适合户外活动。"

⏰ 七、定时任务（心跳）5分钟

让小龙虾从"你不叫我不动"变成"主动帮你干活"：

教师实用心跳模板

直接跟小龙虾说：

帮我设置心跳任务：
1. 每天早上7:30 推送今日课表和天气
2. 每天下午5点 提醒我明天有什么课要备
3. 每周五下午 帮我生成本周教学小结
4. 有重要邮件随时通知我
5. 没事不要打扰我
费用提醒：每次心跳消耗一次API调用（几分钱）。30分钟一次 ≈ 每天48次。用免费额度的模型不用担心这个。

🛡️ 八、安全须知5分钟重要

三件必做的事

  1. 不要把端口开放到公网
    确认地址是 127.0.0.1:18789（只有自己能访问），不要改成 0.0.0.0
  2. 保护好 API Key
    API Key = 密码。不要截图发群里、不要上传到任何地方
  3. 定期更新
    openclaw update — 至少每两周一次
⚠️ 全球已有 42,000+ 个小龙虾实例因配置不当暴露在公网，近 1,800 个泄露了 API 密钥和聊天记录。不要成为其中之一。

执行审批机制

小龙虾做高风险操作时会先问你：

  • Allow once（允许一次）— 不确定时选这个
  • Always allow（永远允许）— 确认安全后选这个
  • Deny（拒绝）— 感觉不对就拒绝

🙋 九、Q&A + 动手环节20分钟

常见问题预备

Q: 这跟 ChatGPT 到底有什么不同？

A: ChatGPT 是"你问它答"的网页聊天。小龙虾是真正的执行助手——它能操作你的文件、连接你的工具、出现在你的聊天软件里、主动帮你做事、还有记忆。

Q: 要花多少钱？

A: 软件本身免费开源。AI模型用国内免费额度（Kimi/通义千问/MiniMax）够日常使用。如果用云平台一键部署，基本零成本。

Q: 我完全不懂技术，能用吗？

A: 能！用云平台方案（MiniMax/Kimi），1分钟就能用上。而且大部分配置都可以直接跟小龙虾聊天完成，不需要你写代码。

Q: 数据安全吗？会不会泄露学生信息？

A: 本地部署的数据全在你自己电脑上。但要注意：你发给小龙虾的内容会通过AI模型API处理，不建议发送学生个人隐私数据（身份证号、成绩等）。用它来辅助备课、整理资料、管日程是完全安全的。

Q: 电脑需要一直开着吗？

A: 本地安装需要。如果用云平台方案或云服务器部署，则不需要。

动手练习建议

  • 选一个方案完成安装（推荐云平台方案）
  • 跟小龙虾说"帮我创建 SOUL.md"
  • 试着让它帮你搜索一个备课相关的话题
  • 让它帮你总结一篇文章
  • 设置一个定时任务（如每天推送天气）
