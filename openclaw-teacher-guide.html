1OpenClaw（小龙虾）是什么？

简单说：OpenClaw 是一个可以帮你干活的 AI 助理系统。大家叫它"小龙虾"🦞。

它和 ChatGPT 的区别是：

🗣️ ChatGPT / Claude

只能「聊天」。你问它问题，它给你回答。仅此而已。

🦞 OpenClaw（小龙虾）

能「干活」。帮你搜索资料、整理文档、发邮件、管理日程、分析数据……它有手有脚。

类比理解：ChatGPT 是一个知识渊博的顾问，你问什么他答什么。
小龙虾是一个能干的秘书，不仅能回答问题，还能帮你执行任务。

它的大脑是大语言模型（如 Claude），但通过 Skills（技能系统） 获得了执行能力。就像一个聪明的人，学会了各种工具的使用方法。

2安装三步走

整个过程大约 10 分钟，只需要一台电脑（Windows / Mac / Linux 均可）。

前提条件

  • 安装 Node.js（版本 18+）：访问 nodejs.org 下载安装包，一路「下一步」即可
  • 一个 AI 服务的 API Key（推荐 Claude 或 GPT，后面教怎么获取）

安装步骤

安装 OpenClaw

打开终端（Mac: Terminal / Win: PowerShell），输入以下命令：

# 全局安装 OpenClaw
npm install -g openclaw

初始化配置

安装完成后运行初始化向导：

# 启动配置向导，按提示填写即可
openclaw setup

向导会引导你：选择 AI 模型 → 填入 API Key → 设置工作目录。

启动并开始使用

配置完成后，启动服务：

# 启动 Gateway 服务
openclaw gateway start

# 开始对话
openclaw chat
💡
手机也能用！ 下载 OpenClaw App（iOS/Android），扫码连接电脑上的 Gateway，之后在手机上就能随时和 AI 助理对话。就像加了一个微信好友一样方便。

3核心概念

理解三个关键词，就理解了小龙虾的全部：

概念 类比 作用
🌐 Gateway（网关） 学校的传达室 接收你的消息，分发给 AI 处理。可以连接微信、Telegram 等。
🧠 Agent（智能体） 大脑 理解你的意图，决定要做什么、怎么做。基于大语言模型。
🔧 Skills（技能） 工具箱 / 才艺 具体的执行能力。搜索、写作、数据分析……每个 Skill 是一项技能。
没有 Skills 的小龙虾 = 聪明的实习生（脑子好使，但啥也不会干）
装上 Skills 的小龙虾 = 全能的教学助理（要啥会啥）

安装技能的方法很简单，一行命令：

# 安装一个技能（以搜索为例）
npx clawhub@latest install tavily-search

4教师必装的 5 个技能

从 11,600+ 个技能中精选，专门针对教师的工作场景。按顺序安装。

为什么第一个装它：
  • 11,600+ 技能质量参差不齐，有些可能存在安全风险
  • Clawsec 自动扫描并给出安全评级：✅ SAFE / ⚠️ CAUTION / 🚫 DANGEROUS
  • 相当于给小龙虾装了「杀毒软件」，之后装别的技能都会先过安检
npx clawhub@latest install clawsec

💰 免费额度：每月1000次搜索，日常够用。需要在 tavily.com 注册获取 API Key。

📊 ④ Office-Automation — 办公自动化 效率神器

覆盖日程、邮件、文档、数据四大核心办公场景。老师用得最多的技能。

教师场景：
  • 「帮我把这份成绩单分析一下，找出进步最大的5个学生」
  • 「整理本周教研组会议纪要」

🧠 ⑤ Self-Improving Agent — 越用越聪明 自我进化

让 AI 记住你的偏好和纠正过的错误，下次自动改进。越用越懂你。

实际效果：
  • 你说过「我们用的是人教版教材」→ 它永远记住，以后不再问

  • 你指出过某个写法有误 → 它记录下来，下次不会再犯
npx clawhub@latest install self-improving-agent

5实战场景：你可以直接说这些话

以下场景覆盖了老师日常工作的方方面面。直接复制就能用：

📝 场景一：智能出题

你说：

「帮我出一套初二数学期中测试卷，涵盖一次函数和全等三角形，难度中等偏上，人教版，附答案和评分标准。」

📋 场景二：生成教案

你说：

「根据人教版高一英语 Unit 3 的课文内容，生成一份2课时的教案，包含教学目标、重难点、教学流程和课后作业。加入一个小组讨论环节。」

📊 场景三：成绩分析

你说：

「这是我班上50个学生的数学成绩（Excel文件），帮我分析：平均分、中位数、各分数段分布，找出进步最大和退步最大的5个学生。」

✉️ 场景四：通知撰写

你说：

「帮我写一份家长会通知，时间下周五下午3点，地点多功能厅，议题是期中反馈和下学期安排。语气正式但亲切，再帮我写一个短信精简版。」

📚 场景五：资料整理

你说：

「帮我搜索并整理"初中生时间管理"的资料，做成一份可以发给学生的指南，语言简单易懂，配上一些实用的小技巧。」

🎯 场景六：课堂活动设计

你说：

「帮我设计一个初三化学"酸碱中和反应"的课堂探究活动，需要有实验方案、学生分组建议、记录表格和总结引导问题。」

🎯
提示词黄金公式 角色（你是初中数学老师）+ 任务（出测试卷）+ 要求（难度中等、人教版、附答案）+ 格式（Word格式、分值标注）
越具体，AI 给你的结果越好！

6安全须知

⚠️
最重要的一条 AI 是工具，不是权威。所有生成的内容都需要老师审核后才能使用！

🔐 数据安全

  • 不要上传学生的敏感个人信息（身份证号、家庭住址等）
  • 成绩数据建议脱敏后使用（用学号代替姓名）
  • API Key 是你的账户密钥，不要分享给他人

📋 使用原则

  • AI 生成的教案和试题，务必检查准确性后再使用
  • 不要让 AI 替代你的专业判断 —— 它是助手，不是替代者
  • 安装技能前用 Clawsec 扫描，🚫 DANGEROUS 评级的坚决不装
  • 初次使用建议在非关键场景练手，熟练后再用于正式工作

7常见问题 FAQ

Q：我完全不懂编程，能用吗？

能！安装只需要复制粘贴几行命令。日常使用就是打字聊天，和用微信一样简单。遇到问题还可以直接问 AI：「我该怎么做？」

Q：和直接用 ChatGPT 有什么区别？

ChatGPT 只能对话。小龙虾能对话 + 干活（搜索、写文件、发邮件、分析数据……）。而且可以连接微信，随时在手机上使用。

Q：学生的数据会不会泄露？

OpenClaw 运行在你自己的电脑上，数据不经过第三方。只有对话内容会发送到 AI 模型提供商（如 Anthropic），它们有严格的数据保护政策。建议敏感数据脱敏后使用。

Q：需要一直开着电脑吗？

想通过手机随时使用的话，需要电脑开着。也可以用一台旧电脑或树莓派专门跑，功耗很低。

Q：API Key 怎么获取？

推荐 Claude：访问 console.anthropic.com 注册 → API Keys 页面创建。需要绑定信用卡，按使用量扣费。也可以用国内的中转服务，更方便。

Q：办公室多位老师可以共用吗？

可以！一个 Gateway 可以连接多个用户。每位老师用自己的微信/Telegram 连接，AI 会分别记住各自的偏好。

Q：AI 出的试题会不会有错误？

会。AI 目前在数学计算和专业知识上还会犯错。所有内容必须人工审核。把它当成一个效率工具，不是万能工具。
